Ao adicionar arquivos .pdf em uma `pasta(Configurável)` e executar o script, ele adiciona os dados ao PDF no local designado em todos os arquivos presentes naquela pasta e os salva em uma nova pasta com a sua nova versão. O script também faz comparação de texto dentro do PDF com uma planilha base do excel para poder validar em qual PDF adicionar cada informação


Utilizado para inserir o CPF de forma automatizada no Holerite -RH
